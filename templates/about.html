About the Project

The Palette & the Stack

Arctic Nord is a minimal Flask site that demonstrates how the 16-colour Nord palette can be applied to web UI — with a single warm orange pulled from nord12 (#D08770) as the call-to-action accent.

Tech Stack

Layer Technology Notes
Server Python 3 / Flask Routes, Jinja2 templates, JSON API
Templating Jinja2 Template inheritance via base.html
Styles Vanilla CSS CSS custom properties, fluid type, 4px spacing scale
JavaScript Vanilla JS (ES2020) Canvas animation, fetch API, Intersection Observer
Fonts Google Fonts JetBrains Mono (display) + Inter (body)
Colour systemNord 0–15 + orange All values in CSS custom properties
Hosting Any WSGI host Gunicorn, Railway, Render, Fly.io, etc.
